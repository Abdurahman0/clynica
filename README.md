# Renaissance Clinic Frontend

Frontend for Renaissance Clinic (admin/developer panel only).

## Source of Truth

Backend/API mapping is defined in:

- `Renaissance Clinic Frontend Pages.txt`

Template documentation from the original Renaissance Clinic clone has been removed.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run typecheck
npm run build
```

## Scope

- Includes: CRM admin/developer pages
- Excludes: Telegram WebApp (separate project)
